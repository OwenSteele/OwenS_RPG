# OwenS_RPG
A text based RPG

VS 2019 C# Ver7.3

To view project - "Open in Visual Studio"

To play the current version of the game:
1. Download .zip file
2. Extract All
3. Navigate to '\WPFUI\bin\Debug'
4. Run 'WPFUI.exe'
5. Enjoy!
